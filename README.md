# Java-Object-Oriented-Programming
## Some projects to use Object Oriented Programming.

### For Maze Solver:
### ->Run it in the terminal, take the input file as an argument.
### ->Program uses right-hand rule on the given grid and prints out coordinates of the exit.
### ->If there's no solution program prints out "No solution."

### For Space Invader Game:
### ->It's a simple game, the spaceship moves to right and left by arrow keys.
### ->It shoots enemies with space key and it may drop two rewards or a penalty.
### ->Rewards are triple shooting or +100 points, and the penalty is -5 points.
